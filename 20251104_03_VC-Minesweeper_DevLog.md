# 2025년 11월 4일 Minesweeper 개발 일지 (3차)

## 작업 요약

게임에 배경 음악 기능을 추가하여 사용자 경험을 향상시켰습니다. 메인 메뉴와 인게임 상태에 따라 적절한 음악이 재생되도록 구현하였으며, 상태 전환 및 게임 종료 시 음악이 자연스럽게 전환되거나 중지되도록 처리했습니다.

## 세부 작업 내용

### 1. Pygame Mixer 초기화 및 사운드 파일 로드

- `pygame.mixer.init()`을 호출하여 Pygame의 사운드 믹서 기능을 초기화했습니다.
- 메인 메뉴 음악 (`VC-Minesweeper_MainMenu.mp3`)과 인게임 음악 (`VC-Minesweeper_Play.mp3`) 파일을 로드하여 `pygame.mixer.Sound` 객체로 준비했습니다.

### 2. 게임 상태에 따른 배경 음악 제어 로직 구현

- **메인 메뉴 음악 재생**: 게임이 시작되어 `main_menu` 상태일 때 `VC-Minesweeper_MainMenu.mp3`가 반복 재생되도록 설정했습니다.
- **인게임 음악 전환**: 사용자가 난이도를 선택하여 `in_game` 상태로 진입할 때:
    - 메인 메뉴 음악을 중지 (`main_menu_sound.stop()`).
    - 인게임 음악 (`VC-Minesweeper_Play.mp3`)을 반복 재생 (`play_sound.play(-1)`).
- **메인 메뉴 복귀 시 음악 전환**: 인게임 상태에서 'Menu' 버튼을 클릭하여 `main_menu` 상태로 돌아올 때:
    - 인게임 음악을 중지 (`play_sound.stop()`).
    - 메인 메뉴 음악을 다시 반복 재생 (`main_menu_sound.play(-1)`).
- **게임 종료 시 음악 중지**: 게임 애플리케이션이 종료될 때 모든 배경 음악이 깔끔하게 중지되도록 `pygame.quit()` 호출 전에 `main_menu_sound.stop()` 및 `play_sound.stop()`을 추가했습니다.

## 결론

배경 음악 추가를 통해 게임의 몰입도를 높이고, 각 게임 상태에 맞는 분위기를 조성할 수 있게 되었습니다. 음악 전환 로직을 통해 사용자에게 더욱 자연스러운 게임 경험을 제공할 것으로 기대합니다.
